## 🚀 Features

- Add daily habits
- Mark habits as completed
- 🔥 Streak tracking system
- 📅 Weekly habit history (7-day view)
- 📊 Progress bar
- 💾 Saves data in browser (localStorage)
- 📱 Mobile app-style UI with bottom navigation
